redefine v2.3.0 【latest】

  1. source 资源文件夹是存放用户资源的地方。除 _posts 文件夹之外，开头命名为 _ (下划线)的文件 / 文件夹和隐藏的文件将会被忽略。Markdown 和 HTML 文件会被解析并放到 public 文件夹，而其他文件会被拷贝过去。
  2. Redefine 主题内置 Fontawesome Pro v6.2.1 版本
  3. 使用 hexo d 命令部署，不要使用 git

安装的插件：

  • hexo-wordcount 字数统计
  • hexo-generator-searchdb 本地搜索
  • nodejieba@2.5.2 推荐阅读板块【安装失败】
  • hexo-all-minifier 静态资源压缩，提升加载速度【安装有漏洞】
  • hexo-deployer-git hexo 自动部署
